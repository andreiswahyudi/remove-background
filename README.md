# remove-background
Background Remover adalah alat otomasi berbasis Python, Streamlit untuk menghapus latar belakang hingga 100 foto secara sekaligus. Proses dilakukan secara lokal untuk menjamin privasi data sepenuhnya. Pengguna cukup mengunggah foto melalui browser dan mengunduh hasilnya dalam satu paket ZIP yang efisien dan praktis.
